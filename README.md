# Living the Simple Life
 
A minimalist blog exploring simplicity and minimalism in everyday life. Built with plain HTML and CSS.
 
## Pages
 
| File | Description |
|------|-------------|
| `index.html` | Home page with a featured article and recent post previews |
| `about-me.html` | About page with the author's personal story |
| `recent-posts.html` | Full listing of all recent posts |
| `style.css` | Shared stylesheet for all pages |
